### Data Analytic Thinking

- 분석적 결정에 따른 데이터의 이해도를 높히기 위해

- 직감만을 믿고 결정하는것이 아닌 데이터 분석에 따라 결정

### The Data Science Process

- KDD(Knowledge Discovery in Database)

  ![KDD](https://github.com/lolhi/ML_Lec/blob/master/MPP/DAT203.1x-Data%20Science%20Essentials/Module1/KDD.jpg)
  
  1.  데이터 습득
  
  2. 데이터 추출, 클리닝, 주석추가
     - 수 많은 데이터 중 실제로 사용할것을 고른뒤, 사용하기 편하게 변환
  3.  데이터 통합, 집합, 표현
     - 데이터 마이닝에 용이하게 데이터 구조를 변경
  4. 분석 및 모델링
     - 머신러닝 및 데이터 마이닝 사용
  5. 해석
     - 마지막에 어떤 결과를 가지게 되는지, 어떻게 해석되는지 등을 알 수 있음

- CRISP-DM(Cross Industry Standard Process for Data Mining)

  ![CRISP-DM](https://github.com/lolhi/ML_Lec/blob/master/MPP/DAT203.1x-Data%20Science%20Essentials/Module1/CRISP-DM.jpg)

  - KDD 공식화에 상당한 노력을 쏟음
  - 화살표를 통해 프로세스 내 필요한 반복 횟수를 알 수 있음
  - 대표적인 데이터 사이언스 프로세스
  - 모든 단계는 다른 프로세스와 기본적으로 동일  
  - 역방향 화살표 때문에 프로세스가 단 한번으로 끝나지 않음
